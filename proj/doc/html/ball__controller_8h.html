◆ BallState

enum BallState

Declaration of the Ball State Machine.

The state machine toggles the ability of the ball to move along the ground, as well as to jump (whenever it is kicked). Both activities involve a preparation state, the state where the actual action occures, a state that resets the atributes after the action is complete (except atributes that are supposed to change) and a rest state.
