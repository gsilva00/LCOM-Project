Detailed Description

This register:

  • Records source of interrupts;
  • Priority of interrupts: LSRERR then DATARDY then THRRDY then Modem Status;
  • When read, UART freezes all interrupts and indicates the highest priority pending one; When there's an interrupt, bits 3,2,1 are the important ones. The interrupt sources are ordered by priority.
