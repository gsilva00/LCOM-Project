enum PlayerStateJump

Declaration of the Player Jump State Machine.

This state machine toggles the ability of the player to jump. It involves a preparation state, the state where the actual jump occures, a state that resets the atributes after the action is complete (except atributes that are supposed to change) and a rest state.
